
# Cerebrix

Plataforma integral de gimnasia cerebral diseñada para todas las edades.

## Misión
Ofrecer un ecosistema de entrenamiento cognitivo divertido, profesional y 100% privado que potencie las habilidades mentales de nuestros usuarios a través de desafíos diarios y seguimiento de progreso.

## Características Principales
* **40 Juegos Cognitivos**: Divididos en categorías de Lógica, Memoria, Cálculo, Atención y Lenguaje.
* **Seguimiento Personalizado**: Gráficos de rendimiento y estadísticas por categoría.
* **Retos Diarios**: Sesiones adaptativas para mantener el hábito.
* **Privacidad Local**: Todos tus datos se guardan de forma segura en tu dispositivo.
