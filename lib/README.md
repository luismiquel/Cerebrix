# 🧠 Cerebrix

**Cerebrix** es una app gratuita para desarrollar la inteligencia humana a través de retos mentales diarios.

🎯 **Categorías de retos**:
- Lógica
- Memoria
- Atención
- Lenguaje

🧩 Diseñada como un juego, con niveles y progreso.  
🛠️ Desarrollada en **Flutter**.  
🌍 Totalmente gratuita y abierta al público.

## 🚀 ¿Qué hace esta app?

Cada día propone un reto aleatorio por categoría para estimular la mente del usuario. Ideal para entrenamiento cognitivo, educativo o recreativo.

## 📦 Tecnologías usadas
- Flutter
- Dart
- GitHub (repositorio)
